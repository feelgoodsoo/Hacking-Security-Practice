# pydbg

### pydbg란?

- 애플리케이션 해킹이나 리버스 엔지니어링에 많이 사용되는 오픈소스 Python Debugger

### 실습 내용

- BreakPoint Handler 응용 프로그래밍
- Memory Access Violation Hanlder 응용 프로그래밍
- Process Snapshot 응용 프로그래밍
